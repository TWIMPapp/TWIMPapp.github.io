# Ant Castes

To replicate the ants role in the real world, we have three different castes of ants.

| Caste  | Energy | Utility                                                                                                                                             |
| ------ | ------ | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| Queen  | 5      | <ul><li>Leads a colony</li><li>Can breed once a month</li><li>Can see quality of food</li><li>Can create trails</li><li>Can verify trails</li></ul> |
| Drone  | 3      | <ul><li>Can breed once a month</li><li>Can create trails</li><li>Can verify trails</li></ul>                                                        |
| Worker | 2      | <ul><li>Can upgrade to a drone</li><li>Can create trails</li></ul>                                                                                  |
